Operaciones con objetos y herencia

Instrucciones de uso:

-El proyecto fue desarrollado en Eclipse IDE
-Abrir el proyecto
-Ejecutar la clase PrincipalObjetos.java
-Se mostrara el siguiente mensaje en consola

"Valor Actual: 0
Ingresa el valor de un lado nuevo: "

-es necesario introducir un valor entero
-se leera el valor y se mostrara el mensaje siguiente:

"Valor Actual: 4

ELIJA UNA OPCION: 
1.-Calcular area de Cuadrado 
2.-Calcular perimetro de Cuadrado 
3.-Calcular volumen de Cubo 
4.-Calcular perimetro de Cubo 
5.-Cancelar "

-Escoger una opcion del menu para realizar la opcion correspondiente o cancelar para regresar a pedir un valor diferente
-si se escoge una de las operaciones del menu esta se procesara y se mostrara el resultado en consola, ejemplo:

"El perimetro del cuadrado es: 16.0"

y se volvera a pedir escoger alguna opcion del menu al usuario.
-si se escoge cancelar se regresara a pedir nuevamente un valor nuevo al usuario
-El programa seguira pidiendo numeros al usuario para procesarlos hasta que este asi lo desee.
-Para detener el programa simplemente hay que parar la ejecucion del mismo.
