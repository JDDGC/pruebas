Traductor Morse

Instrucciones de uso:

-El proyecto fue desarrollado en Eclipse IDE
-Abrir el proyecto
-Ejecutar la unica clase llamada TraductorMorse.java
-Se mostrara el siguiente mensaje en consola

"Introduzca texto alfanumerico para convertir a morse o solo (. y -) para convertir a texto: "

-se Debe introducir una cadena segun se requiera
-si se quiere convertir de morse a texto se debe introducir texto alfanumerico, es decir solo letras y/o numeros
-si se desea convertir codigo morse a texto se debe introducir codigo morse es decir solo . y -
-el programa reconocera automaticamente si la cadena que se introdujo es codigo morse o texto para asi hacer la operacion correspondiente
-en caso de introducir una cadena no valida, es decir, que no sea solo texto alfanumerico o codigo morse(.-) no se llevara a cabo ninguna operacion y se mostrara el mensaje siguiente en consola:

"Entrada no valida, introduzca texto alfanumerico o solo (. y -) segun el caso "

-al introducir una cadena valida ya sea codigo morse o texto alfanumerico se mostrara en consola la traduccion correspondiente,
ejemplo:

"Texto traducido a morse:

.... --- .-.. .-   . ... - .-   . ...   ..- -. .-   .--. .-. ..- . -... .- "


-despues de mostrar la traduccion de la cadena el programa volvera a pedir otra cadena para su traduccion.
-El programa seguira pidiendo cadenas al usuario para traducirlas hasta que este asi lo desee.
-Para detener el programa simplemente hay que parar la ejecucion del mismo.
